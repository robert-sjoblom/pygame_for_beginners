# Part 1: Learning Python

In these six chapters, you will learn the Python programming language by writing small programs in the terminal. Each chapter introduces one or two big ideas and ends with a project that puts them to use.

By the end of Part 1, you will be comfortable with variables, conditions, loops, functions, lists, and dictionaries -- everything you need to start building games.

| Chapter | Title | What You Will Build |
|---------|-------|---------------------|
| 1 | [Getting Started](chapter_01) | Your first program |
| 2 | [Variables and Values](chapter_02) | A calculator |
| 3 | [Making Decisions](chapter_03) | A number guessing game |
| 4 | [Loops and Repetition](chapter_04) | An improved guessing game |
| 5 | [Functions](chapter_05) | A text adventure |
| 6 | [Lists and Dictionaries](chapter_06) | A quiz game |
