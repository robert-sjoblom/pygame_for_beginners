# Part 2: Making Games with Pygame

Now the real fun begins. You will use everything you learned in Part 1 to build graphical games with Pygame -- a library that lets Python draw graphics, play sounds, and respond to keyboard and mouse input.

Each game is more ambitious than the last, and each one teaches you new techniques. By the end, you will have built five complete games.
