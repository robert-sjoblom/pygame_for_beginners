# Making Games with Python and Pygame

*A free book for beginners who want to learn programming by building real games.*

---

## What Is This Book?

This book teaches you programming from scratch using Python and Pygame. You will start with zero experience and finish having built five complete games.

No prior knowledge required. All you need is a computer running Debian Linux, some curiosity, and a willingness to type things in and see what happens.

## Who Is This For?

You, if you have never written a program before. By the end, you will understand variables, loops, functions, lists, classes, and how to build graphical games with keyboard and mouse input, collision detection, and game states.

## How to Read This Book

Read the chapters in order -- each one builds on the ones before it. **Type the code yourself** rather than copying and pasting. Run your code often. Do the exercises at the end of each chapter.

Every chapter is broken into 15-20 minute sections with clear stopping points. You do not need to finish a whole chapter in one sitting.

## The Games You Will Build

| Game | Chapter | What You Will Learn |
|------|---------|---------------------|
| Pong | 8 | Movement, keyboard input, collision detection |
| Snake | 9 | Grid movement, lists as data, game states |
| Flappy Bird | 11 | Gravity physics, spawning objects, sprites |
| Space Adventure | 13 | Mouse input, rooms, inventory, puzzles |
| Tower Defense | 14 | Pathfinding, targeting, waves, game balance |

## Structure

The book has two parts:

**[Part 1: Learning Python](book/part1)** (Chapters 1-6) teaches you the programming language through small text-based programs.

**[Part 2: Making Games with Pygame](book/part2)** (Chapters 7-15) uses everything you learned to build real graphical games.
